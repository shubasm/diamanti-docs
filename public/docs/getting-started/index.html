Diamanti D-Series quick start guide

The Diamanti D-Series Quick start guide introduces the Diamanti D-Series appliance and provides an overview of the principal components and features of the Diamanti software. This guide then describes how to perform the most common operations, including how to create a cluster, define Diamanti objects such as networks and storage volumes, and deploy workloads using Kubernetes.
